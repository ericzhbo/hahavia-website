# 浙江三友发展集团有限公司 - 网站架构规划

## 1. 项目概述

### 1.1 项目背景
浙江三友发展集团有限公司是一家集外贸进出口、房地产开发、金融投资、化工工贸于一体的企业集团，成立于1997年。公司主要经营毛纺等各类面料和服装的生产和贸易，产品远销全球30多个国家和地区。

### 1.2 项目目标
- 构建一个英文企业网站，展示公司形象和产品信息
- 开发一个中文管理后台，用于管理产品资料
- 确保网站视觉效果出色，用户体验良好
- 保持技术栈轻量化，适合小型企业网站

## 2. 网站结构规划

### 2.1 前端英文网站

#### 2.1.1 页面结构
1. **首页**
   - 公司简介
   - 产品展示（卡片瀑布流）
   - 工厂展示
   - 联系方式

2. **关于我们**
   - 公司简介（使用提供的资料）
   - 公司图片展示

3. **产品中心**
   - 产品分类
   - 产品卡片瀑布流展示
   - 点击产品卡片弹出产品详情

4. **工厂展示**
   - 工厂环境图片展示

5. **联系我们**
   - 公司地址
   - 联系电话
   - 邮箱
   - 联系表单

#### 2.1.2 技术选型
- **框架**：纯HTML5 + CSS3 + JavaScript
- **CSS框架**：Tailwind CSS v3（通过CDN引入）
- **图标**：Font Awesome（通过CDN引入）
- **响应式设计**：使用Tailwind CSS的响应式类
- **构建工具**：无需构建工具，直接使用静态文件
- **交互效果**：使用原生JavaScript实现弹出式产品详情

### 2.2 管理后台

#### 2.2.1 功能模块
1. **登录页面**
   - 用户名/密码登录

2. **产品管理**
   - 产品列表
   - 添加产品
   - 编辑产品
   - 删除产品

3. **订单管理**
   - 订单列表
   - 订单详情

4. **内容管理**
   - 网站内容编辑
   - 图片管理

5. **用户管理**
   - 用户列表
   - 添加用户
   - 编辑用户
   - 删除用户

#### 2.2.2 技术选型
- **框架**：纯HTML5 + CSS3 + JavaScript
- **CSS框架**：Tailwind CSS v3（通过CDN引入）
- **图标**：Font Awesome（通过CDN引入）
- **响应式设计**：使用Tailwind CSS的响应式类

### 2.3 后端API

#### 2.3.1 功能模块
1. **产品API**
   - 获取产品列表
   - 获取产品详情
   - 添加产品
   - 更新产品
   - 删除产品

2. **订单API**
   - 获取订单列表
   - 获取订单详情
   - 添加订单
   - 更新订单状态

3. **用户API**
   - 用户登录
   - 获取用户列表
   - 添加用户
   - 更新用户
   - 删除用户

#### 2.3.2 技术选型
- **语言**：Node.js
- **框架**：Express.js
- **数据库**：SQLite（轻量级文件数据库）
- **认证**：JWT
- **CORS**：cors中间件

## 4. 技术架构优势

### 4.1 前端优势
- **轻量化**：使用纯HTML5 + CSS3 + JavaScript，无需复杂框架
- **响应式**：使用Tailwind CSS实现响应式设计
- **快速加载**：通过CDN引入CSS和图标库，减少服务器负担
- **易于维护**：代码结构清晰，易于理解和修改
- **交互友好**：使用弹出式产品详情，提升用户体验

### 4.2 后端优势
- **轻量级**：使用Express.js和SQLite，适合小型应用
- **快速开发**：Express.js提供简洁的API开发体验
- **易于部署**：SQLite是文件数据库，无需额外数据库服务
- **安全性**：使用JWT进行身份验证，保护API安全

## 5. 部署方案

### 5.1 前端部署
- 将静态文件部署到任何静态文件服务器
- 支持GitHub Pages、Netlify、Vercel等托管服务

### 5.2 后端部署
- 部署到Node.js支持的服务器
- 支持Heroku、Vercel、AWS Lambda等托管服务

## 6. 开发计划

### 6.1 阶段一：前端开发

1. 前端与后端集成

## 7. 总结

本架构规划采用轻量化技术栈，适合小型企业网站的需求。前端使用纯HTML5 + CSS3 + JavaScript，结合Tailwind CSS实现响应式设计和卡片瀑布流效果；后端使用Node.js + Express.js + SQLite，提供轻量级的API服务。这种架构不仅开发成本低，而且维护简单，适合中小企业使用。
